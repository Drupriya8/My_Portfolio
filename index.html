My Skills

HTML

HTML stands for Hypertext Markup Language, which is the standard markup language used to create web pages. It uses tags to describe the content and structure of a web page, and these tags are interpreted by web browsers to display the page in a user-friendly format.

CSS

CSS stands for Cascading Style Sheets, which is a style sheet language used for describing the presentation and layout of web pages. CSS is used to control the visual appearance of HTML elements on a web page, such as the color, font, size, layout, and spacing.

Java-script

JavaScript is a programming language used to create dynamic and interactive web pages. It is one of the three core technologies of the World Wide Web, along with HTML and CSS.

React

React is a popular JavaScript library used for building user interfaces (UIs). It was developed by Facebook and is now widely used by developers and companies around the world.

Redux

Redux is a popular state management library used in JavaScript applications, particularly those built with React. Redux provides a centralized store for managing the state of an application, making it easier to keep track of data and to update the UI in response to changes in that data.

Node.Js

Node.js is a popular runtime environment for running JavaScript code outside of a web browser. It allows developers to build server-side applications and command line tools using JavaScript.

Express

Express is a popular web framework for Node.js that provides a set of tools for building web applications. It includes features like routing, middleware, and templating engines, which make it easier to build complex web applications with Node.js.

MongoDB

MongoDB is a popular NoSQL document-oriented database that is designed to store and manage large volumes of data. Unlike traditional relational databases, which store data in tables with predefined schemas, MongoDB stores data in flexible documents with dynamic schemas.

Firebase

Firebase is a backend-as-a-service (BaaS) platform that provides a suite of tools for building web and mobile applications. It is owned by Google and provides a variety of services for building scalable and secure applications without the need for server-side infrastructure.

Bootstrap

Bootstrap is a popular front-end framework for building responsive and mobile-first web applications. It includes a set of CSS and JavaScript components and provides a powerful grid system that makes it easy to create flexible and customizable layouts.

Material UI

Material UI is a popular React UI framework that provides a set of reusable and customizable components for building modern web applications. It is built on top of Google's Material Design guidelines and provides a clean and consistent design system that can be easily customized to fit the needs of your application.

MERN

MERN is a popular web development stack that consists of four technologies: MongoDB, Express.js, React, and Node.js.
